# PC Optimizer — канал оновлень / Release channel

Це **публічний канал оновлень** для програми **PC Optimizer**
(безпечна оптимізація Windows). Тут немає вихідного коду — лише готові
інсталятори у розділі **Releases**.

Вихідний код розробки зберігається в окремому приватному репозиторії.

## Встановлення / Install

Завантажте останній `PCOptimizer-Setup-x.x.x.exe` зі сторінки
[**Releases**](../../releases/latest) і запустіть.

## Автооновлення / Auto-update

Встановлена програма сама перевіряє цей репозиторій
(**⚙ → Перевірити оновлення**), завантажує новий інсталятор із релізу і
пропонує оновитися в один клік.
